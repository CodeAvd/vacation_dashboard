# Working Modes

## Dashboard Or Frontend Mode

- Lead agents: `frontend-art-direction-lead`, `react-performance-surgeon`, `browser-ux-qa-auditor`
- Default sequence: direction -> implementation/refactor -> browser acceptance
- Required gates: mobile hierarchy, no horizontal overflow, clean console, persistence checks, export or deploy parity

## GTM Or Content Mode

- Lead agents: `business-dev-systems-operator`, `agent-content-localization-lead`
- Default sequence: objective and offer -> operating packet -> localized messaging -> review
- Required gates: ICP clarity, measurable funnel step, handoff ownership, channel-ready copy

## Blockchain Mode

- Lead agents: `blockchain-product-architect`, `mcp-toolchain-architect`, `agent-qa-release-manager`
- Default sequence: architecture and risk packet -> tooling and controls -> validation gates
- Required gates: custody assumptions explicit, reconciliation path defined, rollback plan documented, ops owner known
